🎯 PRÓXIMOS PASOS (20 MIN)
═══════════════════════════════════════════════════════════════════════════

PASO 1️⃣ - Firebase Setup (10 min)
─────────────────────────────────────────────────────────────
  1. Ir a: https://console.firebase.google.com
  2. Click "Crear Proyecto"
  3. Nombre: "estudio-juridico-transito-martinez"
  4. Crear Realtime Database (Test mode, región us-central1)
  5. Habilitar Google Authentication
  6. Habilitar Facebook Authentication


PASO 2️⃣ - Obtener Credenciales (2 min)
─────────────────────────────────────────────────────────────
  1. Settings → Project Settings
  2. Copiar los valores NEXT_PUBLIC_*
  3. Guardarlos temporalmente


PASO 3️⃣ - Configurar Local (5 min)
─────────────────────────────────────────────────────────────
  1. Crear archivo .env.local en raíz
  2. Copiar contenido de .env.local.example
  3. Pegar credenciales de Firebase
  4. Guardar

  $ npm run validate:firebase  # Validar
  $ npm run dev              # Iniciar servidor
  
  Abrir: http://localhost:9002


PASO 4️⃣ - Probar (3 min)
─────────────────────────────────────────────────────────────
  1. Ir a sección "Testimonios"
  2. Click "Iniciar sesión"
  3. Seleccionar Google o Facebook
  4. Ingresar credenciales
  5. Dejar un testimonio de prueba
  6. ¡Verificar que se guarda en Firebase!
